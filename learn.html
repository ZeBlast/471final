Dataset

This page aggregates College Scorecard institution-level rows from data/college_scorecard_data/Most-Recent-Cohorts-Institution.csv — the same file that powers Explore. Earnings use published medians after completion (years 1, 5, and 10 where reported); debt uses median graduate debt for completers (GRAD_DEBT_MDN) and overall median debt (DEBT_MDN) where you pick that metric. Many more schools appear here than in the earlier PSEO-only extract, so state medians are more representative.

Learn: State context from Scorecard data

Compare states using the same institution universe as the map: median earnings horizons, debt, cost, and efficiency. Sort and swap metrics to see which regions cluster with stronger earnings relative to borrowing.

Because the underlying file lists most Title IV schools with location and outcome fields, sparse regions on the old PSEO-only map are filled in here. Use Matched college count as the sort key when you want states ranked by how many institutions contributed to the average — higher counts usually mean more stable medians.

When you are ready to inspect individual campuses, open Explore. To compare specific schools side by side, use Decide on the Explore page.
